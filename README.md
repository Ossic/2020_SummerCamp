# 2020 Xilinx Summer Camp

本次由信息技术新工科联盟主办，Xilinx、东南大学，西南交大联合承办的暑期学校已于8月3日落下帷幕。从7月20日到8月3日这15天时间里，超过1200名学员在全世界各地通过互联网参与了这一次暑期学校。

本仓库用于记录本次暑期学校培训的部分项目和日程，详细的培训内容与优秀项目信息请关注下方**PYNQ开源社区**公众号获取。
